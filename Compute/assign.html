赋值运算

赋值运算

  • 赋值号=用于将数据写入变量。

    对变量的操作有两种：读取和写入，而写入时只能用赋值运算。

  • 赋值不是等于。

  • 赋值号两边的数据类型是一致的。
int a = 1;
a = a + 1; // 把a的值取出来，加1，再赋值给a。
Console.WriteLine(a); // 输出：2

表达式

  • 包含赋值运算的语句叫表达式。
  • 单独的运算是不能做为语句的。
// 1+ 2; // Error:不是一个语句。
int a = 1 + 2; // OK：表达式可以做为一个语句。
