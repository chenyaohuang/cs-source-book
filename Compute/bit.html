位运算

  • 位运算是用数据的二进制进行运算。
  • 按位异或，相异为1，相同为0。
  • 左移一位相当于把数据乘以2。

假设变量 A 的值为 60，二进制为0011 1100，变量 B 的值为 13，二进制为0000 1101，

运算符 描述 实例
& 按位与 A & B => 0000 1100
按位或 A B => 0011 1101
~ 按位非 ~A => 1100 0011
^ 按位异或 A ^ B => 0011 0001
<< 按位左移 A << 2 => 1111 0000
>> 按位右移 A >> 3 => 0000 0111
