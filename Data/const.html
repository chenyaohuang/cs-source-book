常量

  • 常量一个不能改变的数据。
  • 在变量的申明前加上const关键字，就可将其变为常量。
  • 常量命名通常用全大字字母，单词间用_隔开。
  • 常量必需在申明时初始化。
    // 常量申明的格式：
// const 数据类型 常量名 = 值;
const double PI = 3.1415927;
PI = 3.14; // 错误。常量不可修改。
