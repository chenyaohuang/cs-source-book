函数的定义

  • 函数的作用：模块化、复用。
  • 函数用来表示一个行为。
  • 函数用来整合一段代码。

之前讲过一组运算表达式可以表示行为。但函数可以用来整合这组运算语句。整体来说一个函数就可以表示一个行为。

void Func()
{

}

函数的调用

  • 调用一个函数就是将函数中的代码从头到尾执行一遍。
  • 函数可以反复调用。

局部变量与全局变量

  • 变量的作用域指在哪个范围内可以访问到这个变量。
  • 变量的作用域通常是从申明开始到包含它的那对{}结束。
  • 变量的生命周期指从变量产生到消亡的时间。
  • 变量的生命周期与作用域通常是一致的。只不过他们一个是时间概念，一个是空间概念。
  • 变量产生时会分配对应大小的内存，消亡时会回收这块内存。
  • 变量消亡后，其存储的数据就一同消亡了。
// global是全局变量，在每个函数中都可以访问。
int global = 100; 

void Fun() 
{
  // a是局部变量，只能在当前这个函数中访问。
  int a = 1; // a的作用域从这里开始
  for(int i = 0; i < 10; i++) // i的作用域从这里开始
  {

  } // i的作用域到这里结束

  // 这里就访问不到i了

  for(int i = 0; i < 5; i++) // 这个i和之前的i不冲突，因为作用域不一样。
  {

  }

} // a的作用域到这里结束

函数的参数

  • 为了保证代码的灵活性，可以将代码中变化的部分提取出来，用变量代替。
  • 局部变量只能在函数内访问，无法从外部给它赋值。
  • 参数是一种特殊的局部变量，它可以通过函数的调用，从外部给参数赋值。
  • 默认值

函数的返回值

  • 如果有返回值必须return。而且是每个分支都有return。
  • return的类型要一致。
  • 一旦return就不再继续执行函数了。
  • 如果函数返回int，那么就可以把函数的调用当作一个int的值来用。

函数的重载

  • 两个函数可以同名，这叫函数的重载。
  • 当调用重载的函数时，要能明确区分调用的是哪个函数。
  • 只能通过函数的参数类型和参数个数来区分函数的重载。

递归调用

  • 我们可以调用任何已知函数
  • 如果调用函数自身，叫递归。
  • 递归能实现一个循环。
  • 递归的哲学：越来越接近已知。
  • 调用自身函数与调用其他函数并没有什么不同，只要明确函数的功能就可以了。

总结

总结下函数中最重要的几点：

  • 函数是一个独立的功能。int Add（int[] data）{ }
  • 调用函数，就是把函数体中的代码从头到尾的执行一遍。
  • 函数返回xx类型，那么就可以把这个函数当作xx类型的数据来用。
