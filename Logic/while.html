循环逻辑-while

语法规则

while(循环条件判断)
{
  // 循环体
}

// 先执行while中的逻辑运算。如果是true，就执行一遍循环体。如果是false，就不执行循环体了。
// 执行完循环体后，会再次判断while中的逻辑运算。

应用案例

  • 死循环。如果while中的逻辑运算永远为true，将会出现无法退出的循环。
while(true)
{
  Console.WriteLine("这是一个死循环");
}
  • 打印3遍
int i = 0;
while (i < 3)
{
  Console.WriteLine("重要的事说三遍。");
  i++;
}
  • 打印0~100
int i = 0;
while (i <= 100)
{
  Console.WriteLine(i);
  i++;
}

跳出循环

  • continue结束本轮循环，循环体中continue之后的代码就不再执行了，继续下一轮循环。
  • break结束循环，整个循环遇到break就结束了，本轮循环中break之后的代码也不再执行了。
  • 在switch-case中，每一个case后用的也是break。

使用场景

从编程思想的角度讲以下两情况可以使用while循环。

  • 只要...就... 。只要条件满足就执行循环体。
// 猜商品的价格
Random r = new Random()
int cost = r.Next(100, 1000); // 商品的价格在100~1000之间
int guess = -1; // guess用来存用户猜的价格

// 只要没猜对，就接着猜。
while (guess != cost)
{
  guess = Int32.Parse(Console.ReadLine());
  if (guess > cost) 
  {
    Console.WriteLine("大了");
  }
  else if (guess <cost)
  {
    Console.WriteLine("小了");
  }
}

Console.WriteLine("猜对了");
  • 直到...才...。
// 猜商品的价格
Random r = new Random()
int cost = r.Next(100, 1000); // 商品的价格在100~1000之间
int guess = -1; // guess用来存用户猜的价格

// 直到...才...。使用死循环。
while (true)
{
  guess = Int32.Parse(Console.ReadLine());

  // 直到猜对了才退出
  if (guess == cost)
    break;

  if (guess > cost) 
  {
    Console.WriteLine("大了");
  }
  else if (guess <cost)
  {
    Console.WriteLine("小了");
  }
}
